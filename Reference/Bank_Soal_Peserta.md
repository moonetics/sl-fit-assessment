# Bank Soal Assessment — Participant Version
# Community Fit Assessment

**Instruksi untuk peserta:**  
Pilih jawaban yang paling menggambarkan kebiasaan kamu di komunitas online. Ini bukan psikotes klinis. Tidak ada jawaban yang “sempurna”. Jawab dengan jujur sesuai perilaku kamu biasanya.

## Skala
1 = Sangat tidak setuju  
2 = Tidak setuju  
3 = Setuju  
4 = Sangat setuju  

---

## Soal Skala 1–4

1. Saya menjaga bahasa chat agar tetap sopan walaupun sedang kesal.
2. Saya merasa spam atau mention berulang itu wajar kalau ingin cepat diperhatikan.
3. Saya membaca aturan channel sebelum bertanya atau bertindak.
4. Saya tidak pernah merasa kesal saat kalah dalam game online.
5. Kalau admin menegur saya, saya akan mencoba memahami alasannya dulu.
6. Saya mudah terpancing membalas ejekan dengan ejekan juga.
7. Saya bisa bercanda tanpa menghina kemampuan bermain orang lain.
8. Jika saya kalah race, saya tetap bisa mengucapkan GG atau menerima hasilnya.
9. Kekalahan biasanya membuat saya ingin menyalahkan map, lag, atau pemain lain di chat.
10. Saya suka meningkatkan waktu/skill obby tanpa merusak suasana komunitas.
11. Menjadi yang tercepat lebih penting daripada menjaga hubungan dengan member lain.
12. Saya selalu membaca semua aturan server secara lengkap sebelum mengirim pesan pertama.
13. Member casual tetap punya tempat penting di komunitas walaupun tidak ikut race.
14. Saya merasa member yang lambat sebaiknya tidak ikut event komunitas.
15. Saat ada konflik, saya lebih memilih menyelesaikannya lewat jalur admin/moderator daripada memperpanjang di public chat.
16. Kalau saya merasa benar, saya akan terus membahas masalah itu di channel publik sampai orang lain mengaku salah.
17. Saya bisa mengakui kesalahan jika saya memang melanggar aturan.
18. Kalau banyak orang juga melanggar aturan, saya merasa tidak perlu bertanggung jawab.
19. Saya kadang perlu waktu untuk menenangkan diri setelah race yang mengecewakan.
20. Saya tidak suka membawa masalah pribadi ke banyak channel Discord.
21. Kalau ada konflik, saya biasanya mencari dukungan teman agar pihak lain terlihat salah.
22. Saya tertarik ikut komunitas bukan hanya untuk menang, tetapi juga untuk suasana dan teman main.
23. Saya kemungkinan akan keluar kalau saya tidak cepat dikenal atau tidak masuk leaderboard.
24. Saya bisa menerima keputusan admin walaupun tidak selalu sesuai keinginan saya.
25. Menurut saya, aturan server boleh diabaikan kalau sedang bercanda.
26. Saya pernah ingin membalas komentar yang terasa menyebalkan, walaupun akhirnya tidak saya lakukan.
27. Saya menghindari kata-kata yang bisa memancing flame war.
28. Trash talk keras adalah bagian normal dari kompetisi online.
29. Saya bisa memberi tips kepada pemain baru tanpa membuat mereka merasa bodoh.
30. Saya hanya mau bermain dengan orang yang skill-nya setara atau lebih tinggi dari saya.
31. Saya tetap menghormati pemenang race walaupun saya merasa sebenarnya bisa menang.
32. Jika saya menang, saya boleh sedikit merendahkan lawan karena itu bagian dari hype.
33. Saya senang mengikuti event kompetitif selama aturannya jelas.
34. Saya akan protes keras jika aturan race membuat saya tidak diuntungkan.
35. Saya tidak pernah sekalipun salah paham dengan orang lain di chat.
36. Saya berpikir sebelum mengirim pesan saat emosi.
37. Saya sering menyesal setelah mengirim pesan saat marah.
38. Saya bersedia meminta maaf jika tindakan saya membuat suasana server tidak nyaman.
39. Saya sulit meminta maaf kalau saya merasa niat saya sebenarnya bercanda.
40. Jika ada rumor tentang member lain, saya tidak langsung menyebarkannya.
41. Saya suka tahu drama terbaru di server dan ikut membahasnya.
42. Saya dapat menerima bahwa tidak semua member ingin race serius.
43. Komunitas obby seharusnya memprioritaskan racer cepat dibanding member casual.
44. Saya mau mengikuti masa trial jika itu membantu admin mengenal perilaku saya.
45. Saya bisa berubah pikiran setelah admin menjelaskan aturan dengan baik.
46. Saya tidak suka jika komunitas terlalu banyak aturan onboarding.
47. Jika saya berbeda pendapat, saya bisa menyampaikan dengan tenang.
48. Saya sering merasa perlu membuktikan bahwa saya benar dalam debat online.
49. Saya bisa aktif di komunitas tanpa mencari perhatian berlebihan.
50. Saya merasa admin harus memberi perlakuan khusus kepada racer yang punya skill tinggi.
51. Saya memahami bahwa skill tinggi tidak memberi izin untuk bersikap toxic.
52. Jika saya merasa aturan tidak adil, saya mungkin butuh penjelasan dulu sebelum menerimanya.

## Soal Situasional

53. Kamu kalah tipis dalam race karena melakukan kesalahan di obstacle terakhir. Apa respons terbaikmu?

A. “GG, aku salah di akhir. Next aku coba improve.”  
B. Diam saja dan mencoba lagi tanpa komentar negatif.  
C. Menyindir map atau lag di chat.  
D. Menuduh pemenang beruntung atau curang tanpa bukti.  

54. Member casual bertanya hal dasar yang menurutmu mudah. Apa yang kamu lakukan?

A. Menjawab singkat dan sopan, atau arahkan ke guide.  
B. Membiarkan member lain menjawab.  
C. Menjawab sambil menyindir “masa gitu aja nggak tahu”.  
D. Mengajak orang lain menertawakan dia.  

55. Admin membuat aturan baru untuk race mingguan yang tidak kamu sukai. Apa responsmu?

A. Bertanya alasan aturan dengan sopan dan memberi masukan.  
B. Mengikuti dulu sambil melihat hasilnya.  
C. Mengeluh di public chat berkali-kali.  
D. Mengajak member lain melawan aturan admin.  

56. Ada member yang memancing emosi kamu di Discord. Apa responsmu?

A. Tidak membalas provokasi dan lapor moderator jika perlu.  
B. Mute/abaikan sementara.  
C. Membalas dengan sindiran agar dia berhenti.  
D. Membalas kasar agar dia kapok.  

57. Kamu menang race dan banyak member memujimu. Apa responsmu?

A. Berterima kasih dan tetap menghargai peserta lain.  
B. Merayakan secukupnya tanpa merendahkan.  
C. Menulis “ez” atau “terlalu gampang” di chat.  
D. Mengejek pemain yang kalah.  

58. Kamu melihat temanmu melanggar aturan kecil saat event. Apa yang kamu lakukan?

A. Mengingatkan secara personal atau lapor admin jika perlu.  
B. Tidak ikut melanggar dan membiarkan admin menangani.  
C. Ikut melanggar karena temanmu juga melakukannya.  
D. Membantu menutupi pelanggaran.  

59. Ada rumor bahwa seorang member menyebabkan drama. Kamu belum tahu faktanya. Apa responsmu?

A. Tidak menyebarkan dan menunggu info dari admin/moderator.  
B. Menghindari pembahasan rumor.  
C. Bertanya ke banyak orang untuk mencari detail.  
D. Menyebarkan rumor ke channel lain.  

60. Kamu merasa tidak cocok dengan satu moderator. Apa tindakanmu?

A. Tetap mengikuti aturan dan menyampaikan concern secara private/resmi.  
B. Menghindari interaksi tidak perlu tetapi tetap sopan.  
C. Menyindir moderator itu di public chat.  
D. Mengajak member lain untuk tidak menghormati moderator tersebut.

## Soal Gaya Komunitas

> Soal berikut tetap tampil sebagai pertanyaan biasa untuk peserta. Jangan beri label personality, MBTI, atau profile code di UI peserta.

61. Saya mudah memulai obrolan ringan dengan member baru di server.
62. Saya lebih nyaman mengamati suasana dulu sebelum ikut banyak ngobrol.
63. Saya senang membantu member baru merasa diterima saat pertama masuk.
64. Saya tetap bisa menjadi member yang baik meskipun tidak terlalu sering muncul di chat.
65. Leaderboard dan improve time membuat saya lebih semangat bermain.
66. Saya lebih mencari suasana main santai daripada mengejar posisi tercepat.
67. Saya suka mengulang run untuk memperbaiki detail kecil dalam permainan.
68. Bagi saya, komunitas tetap seru walaupun saya tidak ikut race serius.
69. Saya biasanya mengikuti arahan admin dulu sambil memahami tujuannya.
70. Saya lebih nyaman mengikuti aturan jika alasan di baliknya dijelaskan dengan jelas.
71. Kalau ada aturan baru, saya bisa menyesuaikan diri tanpa banyak debat.
72. Saya suka memberi masukan jika aturan terasa kurang jelas atau kurang adil.
73. Saat emosi, saya cenderung menunggu tenang dulu sebelum merespons.
74. Jika ada masalah, saya cenderung langsung menyampaikan apa yang saya rasakan.
75. Saya lebih memilih membahas konflik secara private atau lewat moderator.
76. Saya mudah menunjukkan ketidaksetujuan ketika suasana terasa tidak adil.
