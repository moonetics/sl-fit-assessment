## Skala Likert 1–4
1 = Sangat tidak setuju  
2 = Tidak setuju  
3 = Setuju  
4 = Sangat setuju  

## Penjelasan Scoring
- **Normal:** skor = jawaban.
- **Reverse:** skor = 5 - jawaban.
- **Situational:** skor sesuai opsi, biasanya 4/3/2/1/0.
- **Honesty/Consistency:** item disisipkan dan tidak diberi label di UI peserta.
